# rawget

![PyPI - Version](https://img.shields.io/pypi/v/rawget?style=flat&logo=pypi&logoColor=%23fff&labelColor=transparent&link=https%3A%2F%2Fpypi.org%2Fproject%2Frawget%2F)
![PyPI - Python Version](https://img.shields.io/pypi/pyversions/rawget?style=flat&logo=python&logoColor=%233776AB)

A lightweight CLI tool for downloading files from a URL.

![PyPI - License](https://img.shields.io/pypi/l/rawget) ![GitHub Actions Workflow Status](https://img.shields.io/github/actions/workflow/status/clxrityy/rawget/ci.yml?branch=main&style=flat&label=CI)

## Features

- [x] YouTube video download support.
- [x] Cross-platform default download directory detection.
- [x] Simple command-line interface and argument parsing.
- [x] Intelligent file selection based on content type and relevance.
- [x] Safe file handling and naming conventions to prevent overwriting and ensure compatibility.
- [x] Works on all major platforms (Linux, macOS, Windows)

### Installation

```bash
pip install rawget
```

#### Usage

```bash
rawget --help
```
